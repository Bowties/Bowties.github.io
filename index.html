My works

These are examples of projects I've worked on in the past, from both school and work!

Motivation Minders Website Redesign

For my final year project in college, I worked with a small team to redesign and develop a working prototype for the Motivation Minders Website. Our solution was created in Visual Studio using HTML, CSS, JavaScript, Jquery, ASP.NET, MVC, SQL and Bootstrap.

Spark

I have developed internal courses and training materials for existing employees and new hires. Mockups and wireframes were developed in Photoshop, and content was created with HTML and CSS.

Newsletters

I was the editor of a monthly newsletter. I was tasked with collecting, editing, and writing articles, as well as organizing, publishing and distributing the newsletter. I also designed a new interactive template for future releases. Articles were created using the Microsoft Office Suite, and the interactive Template was created using HTML, CSS, JavaScript and Bootstrap.

The Hub

The Hub is an internal website that I have developed over the course of a year. I have built the interactive Landing Page as well as many content pieces and web pages. For the main landing page and additional web pages, I used HTML, CSS, JavaScript, Jquery, and Bootstrap.

Brightspace

I have developed a custom video player using HTML and JavaScript, and I have produced a series of training videos that instructed staff on how to create their own content topics. For the videos, I used Windows Movie Maker and the Lync recording software.

Build a Charm

The interactive Build a Charm application is from the Motivation Minders Website. Shoppers are able to choose a locket and drag and drop different charms inside to create their own jewelry! I created this feature of the website using HTML, Javascript and a number of Jquery plugins.
